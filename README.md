# STM32F4-Eval-board
 

this is a eval_board which allows you to explore  
STM32F4 capabilities. with these features:

GSM  (sim800c)

Ethernet physical with DP83848CVV

RS485 (for modbus rtu)

RS232

CAN bus (Controller Area Network)

SPI flash

EEprom
